🧪 Test Database & Persistence System

Panduan testing untuk fitur database localStorage dan auto-login

Test 1: Register User Baru

MANUAL TEST

📋 Langkah-langkah:

  1. Buka aplikasi Investra (index.html)
  2. Klik tab "Daftar"
  3. Isi form:
    • Username: testuser1
    • Password: test123456
    • Konfirmasi: test123456
  4. Klik tombol "Daftar"
  5. Perhatikan notifikasi dan form login

✅ Expected Results:

  • Notifikasi: "✅ Registrasi berhasil! Silakan login."
  • Otomatis pindah ke tab "Login"
  • Username testuser1 sudah terisi di form login
  • Checkbox "Ingat saya" sudah tercentang (default)

Test 2: Login & Auto-Login

MANUAL TEST

📋 Langkah-langkah:

  1. Setelah register (Test 1), isi password di form login
  2. Pastikan checkbox "Ingat saya" tercentang
  3. Klik "Masuk" atau tekan Enter
  4. Perhatikan notifikasi dan dashboard
  5. Refresh browser (tekan F5)
  6. Perhatikan apakah langsung masuk atau tidak

✅ Expected Results:

  • Notifikasi: "✅ Login berhasil! Selamat datang, testuser1!"
  • Masuk ke dashboard aplikasi
  • Setelah refresh: Langsung masuk tanpa login ulang
  • Console log: "✅ Auto-login berhasil untuk user: testuser1"

Test 3: Data Persistence

MANUAL TEST

📋 Langkah-langkah:

  1. Login dengan user yang sudah dibuat
  2. Setor uang Rp 10.000.000
  3. Beli saham BBCA 10 lot
  4. Catat saldo dan portofolio
  5. Tutup browser sepenuhnya
  6. Buka browser lagi
  7. Buka aplikasi
  8. Cek saldo dan portofolio

✅ Expected Results:

  • Auto-login berhasil
  • Saldo masih sama (Rp 10.000.000 - harga BBCA x 10)
  • Portofolio saham BBCA 10 lot masih ada
  • Semua transaksi masih tercatat
  • Data tidak hilang sama sekali

Test 4: Multiple Users

MANUAL TEST

📋 Langkah-langkah:

  1. Register user1, login, setor Rp 5.000.000
  2. Logout
  3. Register user2, login, setor Rp 10.000.000
  4. Logout
  5. Login kembali dengan user1
  6. Cek saldo user1
  7. Logout, login dengan user2
  8. Cek saldo user2

✅ Expected Results:

  • Saldo user1 = Rp 5.000.000 (tidak berubah)
  • Saldo user2 = Rp 10.000.000 (tidak berubah)
  • Data setiap user terpisah
  • Tidak ada data yang tercampur

Test 5: Logout & Session Clear

MANUAL TEST

📋 Langkah-langkah:

  1. Login dengan user
  2. Setor uang, beli saham
  3. Klik tombol "Keluar"
  4. Konfirmasi logout
  5. Perhatikan form login
  6. Refresh browser
  7. Perhatikan apakah auto-login atau tidak

✅ Expected Results:

  • Notifikasi: "✅ Logout berhasil!"
  • Kembali ke form login
  • Setelah refresh: Tetap di form login (tidak auto-login)
  • Session cleared
  • Data user tetap tersimpan (bisa login lagi)

🔧 Console Testing Commands

// Lihat semua user yang terdaftar:

console.log(JSON.parse(localStorage.getItem('bankAppUsers')));

// Lihat user yang sedang login:

console.log(localStorage.getItem('currentUser'));

// Lihat status remember me:

console.log(localStorage.getItem('rememberMe'));

// Lihat data user tertentu:

const users = JSON.parse(localStorage.getItem('bankAppUsers'));

console.log(users['testuser1']);

// Hitung jumlah user terdaftar:

const users = JSON.parse(localStorage.getItem('bankAppUsers'));

console.log('Total users:', Object.keys(users).length);

// ⚠️ HATI-HATI: Clear semua data (reset):

localStorage.clear();

✅ Success Criteria

  • ✓ User bisa register dengan username & password
  • ✓ User bisa login dengan kredensial yang benar
  • ✓ Auto-login berfungsi saat refresh browser
  • ✓ Data tidak hilang saat tutup & buka browser
  • ✓ Multiple users dengan data terpisah
  • ✓ Logout menghapus session (tidak auto-login lagi)
  • ✓ Data user tetap tersimpan setelah logout
  • ✓ Checkbox "Ingat saya" berfungsi dengan baik
