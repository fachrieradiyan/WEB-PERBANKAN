🧪 Test Priority Profile Auto-Update

Panduan testing fitur Priority Account dengan auto-update halaman profil

Test 1: Upgrade dari Dashboard

PENDING

📋 Langkah-langkah:

  1. Buka aplikasi Investra (index.html)
  2. Login atau register dengan akun baru
  3. Pastikan berada di halaman Dashboard
  4. Setor Rp 100.000.000.000 (100 miliar)
  5. Perhatikan modal notifikasi upgrade muncul
  6. Klik tombol "Luar Biasa! 🎊" untuk menutup modal
  7. Klik tab Profil

✅ Expected Results:

  • Modal upgrade muncul dengan animasi
  • Badge di header berubah menjadi "⭐ PRIORITY"
  • Di halaman Profil, badge menampilkan "⭐ PRIORITY Account"
  • Section "Priority Account Info" muncul dengan 4 benefit cards
  • Background section berwarna kuning dengan border gold

Test 2: Upgrade Saat di Profil (Edge Case)

PENDING

📋 Langkah-langkah:

  1. Login dengan akun regular (belum Priority)
  2. Klik tab Profil terlebih dahulu
  3. Perhatikan tidak ada section "Priority Account Info"
  4. Klik tab Dashboard
  5. Setor Rp 100.000.000.000
  6. Perhatikan modal upgrade muncul
  7. Tutup modal, lalu klik tab Profil lagi

✅ Expected Results:

  • Section "Priority Account Info" sudah muncul tanpa perlu refresh manual
  • Badge profil sudah berubah menjadi "⭐ PRIORITY Account"
  • 4 benefit cards terlihat dengan jelas
  • Tidak perlu reload browser untuk melihat perubahan

Test 3: Data Persistence (Reload Browser)

PENDING

📋 Langkah-langkah:

  1. Setelah upgrade Priority (dari Test 1 atau 2)
  2. Tekan F5 atau reload browser
  3. Login kembali dengan akun yang sama
  4. Klik tab Profil

✅ Expected Results:

  • Status Priority masih tersimpan (tidak hilang)
  • Badge "⭐ PRIORITY" muncul di header
  • Badge "⭐ PRIORITY Account" muncul di profil
  • Section "Priority Account Info" muncul otomatis
  • Total deposit masih menunjukkan 100 miliar+

Test 4: Dark Mode Compatibility

PENDING

📋 Langkah-langkah:

  1. Login dengan akun Priority
  2. Klik tab Profil
  3. Klik toggle Dark Mode di header
  4. Perhatikan tampilan section Priority Account Info

✅ Expected Results:

  • Section Priority Account Info tetap terlihat jelas
  • Background berubah dari yellow-50 ke yellow-900
  • Text tetap readable (kontras yang baik)
  • 4 benefit cards tetap terlihat dengan baik

📌 Quick Reference

💰 Threshold Priority:

Rp 100.000.000.000 (100 miliar)

⭐ Badge Text:

⭐ PRIORITY (header) / ⭐ PRIORITY Account (profil)

🎨 Badge Color:

Gradient: yellow-400 → yellow-500 → yellow-600

💾 Data Storage:

localStorage → bankAppUsers → isPriority
