# 🚀 Cara Enable GitHub Pages

## ⚡ Quick Start (5 Menit)

### Step 1: Buka Repository Settings
```
1. Buka: https://github.com/fachrieradiyan/WEB-PERBANKAN
2. Klik tab "Settings" (⚙️)
```

### Step 2: Enable Pages
```
1. Di sidebar kiri, scroll ke bawah
2. Klik "Pages"
3. Di bagian "Source":
   - Branch: pilih "main"
   - Folder: pilih "/ (root)"
4. Klik "Save"
```

### Step 3: Wait & Access
```
1. Wait 1-2 menit untuk deployment
2. Refresh halaman Settings → Pages
3. Lihat URL yang muncul (warna hijau):
   
   ✅ Your site is live at:
   https://fachrieradiyan.github.io/WEB-PERBANKAN/
   
4. Klik URL untuk buka website
```

## 🔍 Troubleshooting

### Jika Halaman Blank:

1. **Wait 2-3 menit** (deployment butuh waktu)
2. **Hard refresh:** Ctrl + Shift + R
3. **Check console:** F12 → Console
4. **Clear cache:** Ctrl + Shift + Delete

### Jika Login Tidak Berfungsi:

1. **Buka console:** F12
2. **Lihat error** (jika ada)
3. **Test di browser lain**
4. **Clear localStorage:**
   ```javascript
   localStorage.clear();
   location.reload();
   ```

### Jika 404 Error:

1. **Check Settings → Pages**
2. **Pastikan Branch = main**
3. **Pastikan Folder = / (root)**
4. **Wait 2-3 menit lagi**
