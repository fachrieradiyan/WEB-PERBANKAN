## 🐛 TROUBLESHOOTING

### Masih 404 setelah enable?
1. **Wait**: First build bisa 5-10 menit
2. **Clear cache**: Ctrl+Shift+R
3. **Incognito**: Buka di private/incognito mode
4. **Check Actions**: https://github.com/fachrieradiyan/WEB-PERBANKAN/actions

### Repository Private?
- GitHub Pages free = public repo only
- Settings → General → Change visibility → Make public

### Build Failed?
1. Go to Actions tab
2. Click latest "pages build and deployment"
3. Check for errors
